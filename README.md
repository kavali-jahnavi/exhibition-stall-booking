# Exhibition Stall Booking System

## 🚀 Description
A web-based application for managing exhibition stall bookings and providing interactive navigation.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Flask (Python)
- SQLite

## ✨ Features
- Stall booking system
- Interactive navigation
- User-friendly interface
- Efficient data management

## 📌 Future Improvements
- Add user login system
- Integrate payment system# exhibition-stall-booking

## ▶️ How to Run

1. Install Python
2. Install Flask:
   pip install flask
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/
